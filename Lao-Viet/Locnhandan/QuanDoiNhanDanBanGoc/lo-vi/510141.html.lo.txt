ທ່ານລັດຖະມົນຕີການຕ່າງປະເທດ ອີຣານ ກໍ່ໄດ້ຕີລາຄາວ່າ “ການມີສະຕິແມ່ນໄດ້ປຽບ ເມື່ອໃຫ້ຄຳເຫັນກ່ຽວກັບເລື່ອງທ່ານ Donald Trump ຍົກເລີກເຈດຈຳນົງບຸກໂຈມຕີໃສ່ອີຣານເມື່ອທ້າຍເດືອນ ມິຖຸນາ ທີ່ຜ່ານມາ.	"Ý thức là một lợi thế khi Donald Trump hủy bỏ ý định tấn công Iran vào cuối tháng 6", ông nói.
ທ່ານ Mohammad Java Zarif ລັດຖະມົນຕີກະຊວງການຕ່າງປະເທດ ອີຣານ (ພາບ: RNA/TTXVN).	Bộ trưởng Ngoại giao Iran Mohammad Java Zarif (Ảnh: RNA / TTXVN)
ວັນທີ 16 ກໍລະກົດ, ເມື່ອໃຫ້ສຳພາດຕໍ່ນັກຂ່າວຊ່ອງໂທລະພາບ ອາເມລິກາ NBC, ທ່ານ Mohammad Java Zarif ລັດຖະມົນຕີກະຊວງການຕ່າງປະເທດ ອີຣານ ໄດ້ປະກາດວ່າ ເຕເຮຣານ ພ້ອມແລ້ວທີ່ຈະປຶກສາຫາລືກັບ ວໍຊິງຕັນ, ຖ້າຫາກວ່າ ອາເມລິກາ ຍົກເລີກຄຳສັ່ງລົງໂທດເພື່ອຕ້ານອີຣານ.	Trả lời phỏng vấn NBC ngày 16/7, Ngoại trưởng Iran Mohammad Java Zarif tuyên bố, Tehran sẵn sàng hội đàm với Washington nếu Mỹ dỡ bỏ lệnh trừng phạt đối với Iran.
ທ່ານ Mohammad Java Zarif ໃຫ້ຮູ້ວ່າ: ທ່ານປະທານາທິບໍດີ ອາເມລິກາ Donald Trump ບໍ່ຢາກດຳເນີນສົງຄາມກັບອີຣານ, ເຖິງຢ່າງໃດກໍ່ຕາມ “ເຈົ້າໜ້າທີ່ຫຼາຍທ່ານທີ່ໃກ້ຊິດກັບປະທານາທິບໍດີ ພັດຕ້ອງການສິ່ງນີ້.	Tuy nhiên, Mohammad Java Zarif nói rằng Tổng thống Mỹ Donald Trump không muốn gây chiến với Iran, “nhiều quan chức thân cận với tổng thống muốn điều này.
ທ່ານລັດຖະມົນຕີການຕ່າງປະເທດ ອີຣານ ກໍ່ໄດ້ຕີລາຄາວ່າ “ການມີສະຕິແມ່ນໄດ້ປຽບ ເມື່ອໃຫ້ຄຳເຫັນກ່ຽວກັບເລື່ອງທ່ານ Donald Trump ຍົກເລີກເຈດຈຳນົງບຸກໂຈມຕີໃສ່ອີຣານເມື່ອທ້າຍເດືອນ ມິຖຸນາ ທີ່ຜ່ານມາ.	"Sự tỉnh táo là một lợi thế khi Donald Trump hủy bỏ ý định tấn công Iran vào cuối tháng 6", ông nói.
ໃນວັນດຽວກັນ, ການນຳສູງສຸດຂອງ ອີຣານ - ສາດສະດາ Ali Khamenei ໄດ້ປະກາດວ່າ ປະເທດ ອິດສະລາມນີ້ຈະສືບຕໍ່ຫຼຸດຜ່ອນບັນດາຄຳໝັ້ນສັນຍາຕາມຂໍ້ຕົກລົງນິວເຄຼຍ, ພ້ອມທັງກ່າວຫາບັນດາຄູ່ຮ່ວມມືເອີລົບທີ່ບໍ່ປະຕິບັດພັນທະຂອງຕົນ.
